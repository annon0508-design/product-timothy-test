# Blueprint: Voice Evidence Finder

## 1. Product Overview

**Product Name:** Voice Evidence Finder  
**Platform:** Web (Firebase-based)  
**Core Function:** Convert uploaded audio files into timestamped transcripts, enable keyword-based search, and generate legally useful PDF reports.

**Positioning:**
A tool to quickly find legally meaningful statements inside recorded audio.

---

## 2. Goals

### Primary Goal
- Allow users to locate specific statements (e.g., promises, agreements, money mentions) inside audio recordings quickly.

### Secondary Goals
- Provide timestamp-based playback for verification
- Export structured PDF for evidence usage

---

## 3. Target Users

- Individuals involved in disputes (money, agreements, fraud)
- Small business owners handling verbal agreements
- Freelancers dealing with client negotiations

---

## 4. Core Features (MVP Scope)

### 4.1 File Upload
- Drag & Drop interface
- Supported formats: mp3, wav, m4a
- Max length: 30 minutes (MVP constraint)

### 4.2 Transcription (STT)
- Use Whisper API (or equivalent)
- Output:
  - Full transcript
  - Timestamps per sentence/segment

### 4.3 Keyword Search
- Input: user-defined keyword
- Output:
  - Matching sentences
  - Corresponding timestamps

### 4.4 Audio Playback
- Built-in player
- Clicking a sentence jumps to timestamp

### 4.5 PDF Export
- Include:
  - Full transcript
  - Highlighted keyword results
  - Timestamps
- Footer:
  "Generated by Voice Evidence Finder"
